Open World Games API

Contains data (JSON) about video games of the Open World genre.

Release year indicates the initial release, Metacritic score is that of the PC version where available.

Use:

https://open-world-video-games-api.onrender.com/api/ - Returns the full list of games.

https://open-world-video-games-api.onrender/api/rating/<rating> - Returns all games in the list with at least the specified Metacritic rating.

https://open-world-video-games-api.onrender/api/platform/<platform> - Returns all games available for the specified platform (PC, Xbox, PlayStation or Switch)

https://open-world-video-games-api.onrender/api/release/<year> - Returns all games released in or after the specified year
