Open World Games API

Contains data (JSON) about video games of the Open World genre.

Release year indicates the initial release, Metacritic score is that of the PC version where available.

Use:

https://xxx.com/api/ - Returns the full list of games.

https://xxx.com/api/rating/<rating> - Returns all games in the list with at least the specified Metacritic rating.

https://xxx.com/api/platform/<platform> - Returns all games available for the selected platform (PC, Xbox, PlayStation or Switch)
